
El proceso de compra tiene 4 partes:
  -primero el usuario envía productos al carrito. 
  -Segundo se procede con la apertura del carrito con el botón Carrito de la barra de navegación.
  -Tercero, se prosigue con la compra presionando Comprar.
  -Cuarto, se depliega formulario y botón Pagar para finalizar la compra.


*Permite agregar productos al carrito de compras.
*Permite quitar productos del carrito.  
*Cada producto tiene un stock definido, luego del cual se visualiza en el producto que no hay stock del mismo.
*Hay un filtro por color y otro filtro precio, no funcionan simultaneamente.
*La entrada del formulario debe respetar formato preestablecido, si se clikea PAgar y sin el formato de input requerido, 
se marca mensaje de error hasta que se complete el campo y se le de enter, o se complete todos con el formato requerido y 
se presiones botón PAGAR.
